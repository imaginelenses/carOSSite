Cars are now at the early stage of smartphones. {% image imgSrc, "featurePhones.png", "", "100vw", "", false, false %} Each company builds their own proprietary software.
Software defined vehicles need software defined companies. Most car companies don’t work this way. {% image imgSrc, "teslaInterior.jpg", "", "100vw", "", false, false %} Tesla is built from the ground-up to be a software company and has the best-in-class software.

Problems

Bad User Interface & lack of options leads users to default to Android Auto & Apple CarPlay. However their favorite maps application does not know the car’s range and their favorite voice assistant cannot set the car's temperature.
Long Time-to-Market as software is not designed to be reused & is not shared between models. Chip shortages will disrupt production as each car has thousands of chips, each running its own software.

Solution

A Revolutionary UI designed for cars.
An Operating System that runs everything from telemetry, ADAS to infotainment. Providing a cohesive user experience and using significantly fewer chips. An App Store for third-party developers to build apps for cars. Developers will have access to data like range and non-crucial controls like temperature.
